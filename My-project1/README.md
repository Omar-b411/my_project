## 🚀 Featured Project

### 🧮 BMI Calculator (Week 02)

A user-friendly Flutter application designed to calculate Body Mass Index with enhanced UI/UX.

#### Key Features:

* Interactive weight & height controls (Slider + Buttons)
* Real-time BMI calculation
* Dynamic category classification:

  * Underweight
  * Normal
  * Overweight
  * Obese
* Visual feedback using adaptive color indicators
* Clean and responsive interface using Material Design principles
